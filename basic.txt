pip install langchain -> to install langchain framework

langchain -> framework allows developers to create variety of applications using language models

Context-aware features: LangChain connects language models with various context sources. This includes prompt directives, a handful of examples, and content based on responses. This allows language models to generate more accurate and relevant answers based on the information provided.

The ability to infer: In addition, the language model can provide any answer based on the context given, or deduce for yourself what action to take. This means that the language model can go beyond simply reproducing information, analyzing a given situation and presenting an appropriate solution.

langchain -> foundation, --> prompts, models, tools, retrievers all are wired together into a clean workflow. used for building simple rag sys, chatbot and simple agents

langGraph -> memory, loop, decision, mult-agent-coordination langchain isn't alone enough. langGraph lets use stateful, branching and agentic workflows

langflow -> drag, drop, experiment, prototype your ai pipelines without writing much code, perfect for fast iteration and collaboration

langsmith -> production mode, tracing, debugging, evaluation, token tarcking everything you need to monitor and improve real ai apps

therefore they all  work together

langchain setup :

create a repo for tracking and learning

name langchain-yourname my case langchain-vip

create a env in conda 

cmd --> conda create -n langchain python=3.11

activate that environment 

conda activate langchain

inside that environment

pip install langchain -->  for installing langchain

pip install langchain langchain-google-genai google-generativeai --> specific for google geminin and genai models

similar cmds used for openai, anthropic, mistral etc
